Vibration Profiling of ball-screw

Components: Accerlometer(ADXL), Encoder, microSD card shield, Arduino.


ML Try1 is MATLAB file to covert data into trainalbe data, use nnstart and start training network on Matlab.
Optimal speed content code for cleaning data (Thresholding data).
DataAquisation folder contain code for reading vibration data and distance(position) with the help of ADXL, Encoders.
DataAquisationAndSaving folder contain code for reading as well as saving the above data into SD card.
SDcardTest contains code for testing SDCard.
